这里是中国科学院电动力学808真题集。

该文档共分为三个文件夹。

第一个是808，这部分文件编译了包含封面，序言等部分。

第二个是readme，包含解释文件，更新日志等。

第三个是test文件夹，主要是试题的编译，Text文件夹中单独以年份命名的文件夹则为该试题集中出现的图片，并已出现顺序依次命名，如pic1，然后是各试题的编译文件。

然后还有一些零碎的文件。

包含试题中的一些图片，以及12年13年，20年的电子档。

对于不会编译的，还可在文件中直接找到电动力学真题集。

另外还上传了网友的一份答案仅供参考。

然后是latex的编译，这里的编译器采用的是XeLaTex，整个文档的架构可在main.tex中查看。

欢迎后来者，对该文档进行修改，补充，完善。
